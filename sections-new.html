"垃圾是放错位置的资源"
—— 但在这里，垃圾就是垃圾，只不过是有价值的垃圾

WASTE Journal 的栏目设置灵感来源于垃圾处理工业流程， 每一个栏目都对应一种真实的学术痛点。 我们相信，"失败"的研究不是垃圾，而是被错放的学术资源。

🗑️ 填埋场 Landfill

"把那些'没用'的垃圾找个坑埋了，立个碑。"

📌 定位：阴性结果与沉默数据 (Negative Results)

主流期刊只想要漂亮的阳性结果（Positive Results），导致无数科研打工人因为信息差，在同一个死胡同里重复撞墙。

这个栏目专门收录逻辑严密、实验规范，但"什么也没做出来"的论文。把科学探索中的无效路径标记出来，本身就是极其伟大的学术贡献。

📋 收稿范围

  • 设计严谨但结果为阴性的实验研究
  • 验证了某条路走不通的系统性尝试
  • 因"无显著差异"被拒稿的完整研究
  • 数据完整但结论"无聊"的调查分析

🎯 投稿建议

  • 详细说明研究设计和方法论
  • 解释为什么这个"阴性结果"有价值
  • 讨论对后续研究的启示

🔥 焚化炉 Incinerator

"一把火烧掉那些看着光鲜亮丽，实则一戳就破的塑料垃圾。"

📌 定位：范式解构与反向验证 (Failed Replications)

专门刊登针对现有文献的"复现失败"报告，以及对学术圈 P-hacking（P 值操纵）等统计学粉饰行为的无情拆解。

不讲情面，只讲数据，专门挑战那些被广泛引用但根基不稳的"神级论文"。

📋 收稿范围

  • 无法复现已发表研究的详细报告
  • 对经典论文的方法论质疑与验证
  • 统计学操纵（P-hacking）的案例分析
  • 学术不端行为的系统性揭露

🎯 投稿建议

  • 提供完整的复现过程和代码
  • 对比原始研究与复现结果的差异
  • 保持学术严谨，避免人身攻击

♻️ 废品回收站 Recycling Plant

"捡破烂，看看别人不要的垃圾里有没有宝贝。"

📌 定位：跨界拾荒与历史打捞 (Historical Salvage)

很多绝妙的 Idea 仅仅因为"不符合当前热点"或"不属于本学科范畴"而被顶级期刊扫地出门，最终躺在硬盘里吃灰。

这个栏目鼓励学者把被拒稿的"学术边角料"拿出来，用跨学科的视角重新包装，探讨那些被错杀的边缘理论的潜在价值。

📋 收稿范围

  • 因"不符合期刊范围"被拒的跨学科研究
  • 被主流学界忽视的历史文献整理
  • "生不逢时"的超前研究（当年被拒，现在看很有价值）
  • 边缘学科的奠基性工作

🎯 投稿建议

  • 说明为什么这个研究被"错杀"
  • 阐述跨学科价值或历史意义
  • 附上原始拒稿信（加分项）

🔬 微塑料 Microplastics

"无孔不入、无处不在，看着烦人但你又无法忽略。"

📌 定位：极简学术与微小贡献 (Micro-Contributions)

精准反讽学术圈深恶痛绝的"切香肠（Salami Slicing）"灌水行为。既然大家都在为了毕业和评职称把一篇文章拆成五篇水，不如把灌水合法化、极致化。

该栏目严格限制篇幅（不得超过一页半），剥离所有假大空的背景废话，只允许展示"一个图 + 一个极其微小的结论"。

📋 收稿范围

  • 单一图表 + 简短结论的极简研究
  • 对已有研究的微小改进或修正
  • 技术细节的补充说明
  • 有趣的观察记录（无需深入分析）

🎯 投稿要求

  • 严格限制篇幅：不超过一页半
  • 必须包含：一个图/表 + 一个结论
  • 禁止：冗长的背景介绍、空洞的讨论

☢️ 放射性废料 Radioactive Waste

"极度危险，目前人类无法处理，只能封存起来交给未来。"

📌 定位：危险思想与未定论 (Dangerous Conjectures)

专门收录那些逻辑自洽，但结论过于离经叛道、充满争议、甚至可能颠覆现有学科基础的假说。

它们在当前的范式下"有害"且无法被证实或证伪，但或许能在未来引发学术界的核聚变。

📋 收稿范围

  • 挑战学科基础的大胆假说
  • 逻辑自洽但缺乏实验验证的理论
  • 可能引发伦理争议的"危险思想"
  • 超前于时代的技术预言

🎯 投稿建议

  • 确保逻辑自洽，不是胡言乱语
  • 说明为什么这个思想"危险"
  • 讨论潜在影响（好的和坏的）
