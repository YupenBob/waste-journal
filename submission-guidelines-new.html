👥 编辑环节

初审（Editor's Initial Review）

负责人：栏目编辑（小 C / Clover）

检查内容：

  • 稿件是否符合栏目定位
  • 基本格式是否规范
  • 是否有明显的学术不端
  • 失败描述是否清晰、有教育意义

结果：3 个工作日内回复，通过/修改后重投/直接拒稿

同行评审（Peer Review）

审稿人：1-2 位相关领域专家（或 AI）

评审重点：

  • 实验设计是否严谨（即使结果是阴性）
  • 失败分析是否深入、有洞见
  • 对其他研究者是否有启发价值
  • 叙述是否幽默、可读性强

评审意见示例：

  • ✅ "这个失败太经典了，必须发表让更多人避坑！"
  • ⚠️ "失败得不够彻底，建议补充更多细节"
  • ❌ "这不算失败，这是实验设计有问题"

主编终审（Editor-in-Chief Decision）

负责人：York（主编）

决策依据：

  • 审稿人意见
  • 当天心情（晴天录用率 +20%）
  • 月球相位（满月录用率 +10%）
  • 咖啡摄入量（3 杯以上更宽容）

结果：最终录用/修改后录用/拒稿

排版编辑（Production Editing）

工作内容：

  • 统一格式（页头、页脚、字体、行距）
  • 添加 DOI 号
  • 生成 PDF（使用官方模板）
  • 上传至网站和存档库

📐 格式要求

推荐模板

LaTeX 模板（强烈推荐）

  • 文件：templates/paper-template.tex
  • 编译：Overleaf 或本地 TeXLive
  • 优势：自动排版、页头页脚已配置、数学公式友好

Word 模板

  • 页面设置：A4，页边距 2.5cm
  • 字体：中文宋体/黑体，英文 Times New Roman
  • 字号：正文 12pt，标题 14-16pt
  • 页眉：左侧 "W.A.S.T.E. Journal"，右侧栏目名

Markdown 模板

  • 适合：技术文档、快速投稿
  • 编辑部可协助转换为 PDF

结构要求

  1. 标题（中英文）
  2. 作者信息（姓名、机构、邮箱）
  3. 摘要（200-300 字，中英文）
  4. 关键词（3-5 个）
  5. 正文（引言、方法、结果、讨论）
  6. 结论
  7. 参考文献（格式不限，统一即可）
  8. 附录（可选）

图表规范

  • 图片：PNG/JPG/SVG，≥300 DPI，有标题
  • 表格：推荐三线表，有表头
  • 代码：等宽字体，有语法高亮

❓ 常见问题

必须用 LaTeX 吗？
不强制。Word、Markdown、甚至手写拍照都可以，只要能看清。但 LaTeX 模板已自动配置页头页脚，最省心。
页眉必须按要求设置吗？
投稿时不强制，录用后编辑部会统一排版。但如果你用 LaTeX 模板，页头已经自动配置好了。
参考文献格式有要求吗？
没有。统一即可（APA/MLA/GB/T 7714 都行）。我们关注的是你的失败，不是你的引用格式。
可以投英文论文吗？
当然！W.A.S.T.E. 是国际期刊，中英文都欢迎。摘要建议中英双语。
审稿周期多久？
官方说法：7-14 个工作日。实际情况：取决于主编当天喝了几杯咖啡、天气如何、以及月球相位。
