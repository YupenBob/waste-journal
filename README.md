# 🗑️ RUBBISH 国际学术期刊

**Research Upon Bizarre & Borderline Incomprehensible Scientific Hypotheses**

**创刊日期**：2026 年 4 月 1 日（愚人节）
**Slogan**："Turning Academic Trash into Treasure"（变废为宝，学术拾荒）

---

## 📋 项目概述

RUBBISH 是一本讽刺性反常规学术期刊，致力于收录全宇宙被主流期刊无情拒绝的、毫无实用价值的、实验完全失败的，以及脑洞大到女娲都补不上的学术论文。

**办刊宗旨**：
> 在这个内卷的时代，不是每篇论文都能改变世界，但至少，你的论文可以改变读者的心情。

---

## 🎯 目标受众

- 常年被"审稿人 2 号 (Reviewer #2)"折磨到精神衰弱的科研工作者

- 需要找点乐子来缓解发际线后移焦虑的硕博研究生

## 🎲 评审机制

我们坚决摒弃传统的双盲评审，采用独创的评审方式：

- **🎯 盲盒评审**：审稿人扔飞镖或掷骰子决定
- **😂 情绪价值评审**：取决于主编笑出多大声
- **🍻 双醉评审**：酒吧微醺后讨论，双方满意就录用
