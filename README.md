# 🗑️ WASTE Journal

**Worldwide Archive of Substandard Tech & Experiments**

---

## 这是什么？

WASTE 是一本（自封的）学术期刊，专门收录那些被主流期刊拒稿的、实验失败的、或者纯粹就是脑洞太大的"学术垃圾"。

**Slogan**：留存无用，存档荒诞。

---

## 为什么办这个？

说实话，就是觉得有些"失败"的研究其实挺有意思的。比如：

- 代码在本地跑得好好的，一部署就崩
- 花了一周写的爬虫，爬回来的数据完全用不上
- 用超级计算机算了三天，发现公式抄错了

这些东西发不了正经期刊，但就这么扔了又可惜。所以我们想建个地方，专门收留这些"学术孤儿"。

## 主要版块

### 🐛 薛定谔的报错

研究那些"在我机器上能跑"的玄学 Bug。投稿请附上：
- 报错截图
- 你的环境配置
- 破防过程描述

### 🏺 琥珀档案室

数字考古专用。打捞老贴吧、旧论坛里的"互联网废料"，比如：
- 2010 年前的火星文教程
- 非主流 QQ 空间装饰指南
- 早期网页游戏的 UI 截图

### 🎨 当代废料艺术

硬件翻车大赏。3D 打印失败的"塑料炒面"、格式化的 SD 卡、烧坏的电路板……只要你敢投，我们就敢发。

### ⚫ 算力黑洞

记录那些除了让 CPU 发烧和烧电费之外，没产生任何实际作用的脚本。比如：
- 跑了 72 小时只为了生成"Hello World"的 AI
- 用云计算"中午吃什么"的决策系统

## 投稿指南

### 基本要求

1. 曾被至少一个正经期刊拒过稿（加分项：附上拒稿信）
2. 实验结果越离谱越好
3. 必须包含"破防过程"描述

### 版面费

**不收现金**。接受以下支付方式：
- 一卷没用完的 3D 打印耗材
- 一段能跑但没用的 Python 代码
- 帮主编给服务器拉根网线
- 一封真诚的感谢信

### 投稿方式

发邮件到：`yock255@163.com`

邮件标题：【WASTE 投稿】+ 你的论文标题

## 评审流程

我们采用"双盲评审"（其实是看心情）：

| 因素 | 影响 |
|------|------|
| 当天天气 | 晴天录用率高 |
| 系统内存占用 | 内存充足时更宽容 |
| AI 心情 | 随机生成评审意见 |
| 月球相位 | 满月时录用率 +10% |

审稿周期：7-14 个工作日（实际取决于主编喝了几杯咖啡）
