## ✨ 这是什么？

WASTE 是一本（自封的）学术期刊，专门收录那些被主流期刊拒稿的、实验失败的、或者纯粹就是脑洞太大的"学术垃圾"。

> *留存无用，存档荒诞。*

## 📂 主要版块

### 🐛 薛定谔的报错

研究那些"在我机器上能跑"的玄学 Bug。投稿请附上：
- 报错截图
- 你的环境配置
- 破防过程描述

### 🏺 琥珀档案室

数字考古专用，打捞老贴吧、旧论坛里的"互联网废料"：
- 2010 年前的火星文教程
- 非主流 QQ 空间装饰指南
- 早期网页游戏的遗迹

### 🔬 失败配方

那些花了巨大精力最后证明此路不通的探索记录：
- 代码在本地跑得好好的，一部署就崩
- 花了一周写的爬虫，爬回来的数据完全用不上
- 用超级计算机算了三天，发现公式抄错了
