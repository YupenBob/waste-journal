Submission Guidelines

Before You Submit

请确认你的研究符合以下标准：

  • ✅ 实验真实存在，失败真实发生
  • ✅ 能让后来者避免同样的坑
  • ✅ 破防过程描述生动
  • ✅ 能让人会心一笑
⚠️ 注意： 如果你的研究过于成功，请改投其他期刊。WASTE 只收"失败"研究。

Submission Process

1

准备稿件

格式要求：

  • 标题：清晰描述你的"失败"
  • 摘要：200-300 字，说明问题、方法、（失败的）结果
  • 引言：为什么你觉得这个研究有意义
  • 方法：你具体做了什么
  • 结果：实际发生了什么（越离谱越好）
  • 破防过程：从自信到怀疑人生的心路历程
  • 结论：你学到了什么（或者没学到）
  • 参考文献：任何对你有启发（或误导）的资料

接受格式：LaTeX / Word / Markdown / 手写拍照（只要能看清）

2

发送邮件

收件人：waste-journal@example.com

邮件标题：【WASTE 投稿】+ 你的论文标题

邮件正文：

  • 作者姓名（或笔名）
  • 联系方式（邮箱即可）
  • 投稿版块（薛定谔的报错/琥珀档案室/当代废料艺术/算力黑洞）
  • 一句话总结你的"失败"

附件：论文文件（PDF 优先）+ 补充材料（可选）

3

等待评审

审稿周期：7-14 个工作日（实际取决于主编当天喝了几杯咖啡）

评审结果：

  • ✅ Accept：恭喜！你的"失败"被认可了
  • 🔄 Revise：需要补充更多"破防细节"
  • ❌ Reject：你的失败还不够有教育意义（或不够好笑）
4

支付版面费

APC（Article Processing Charge）：

本刊不收现金。接受以下支付方式：

  • 一卷没用完的 3D 打印耗材
  • 一段能跑但没用的 Python 代码
  • 帮主编给服务器拉根网线
  • 一封真诚的感谢信
5

发表

论文将在 WASTE Journal 网站发表，分配 DOI 号，进入永久存档。

注意：一旦发表，你的"失败"将成为人类知识的一部分，被后人引用（或嘲笑）。

Ethics

学术诚信：

  • 研究必须真实，不得伪造数据
  • 可以夸张描述，但不能无中生有
  • 引用他人"失败"需注明出处

一稿多投：

  • 允许一稿多投（反正大概率被拒）
  • 但如果被 WASTE 录用了，请第一时间通知我们
